Java Projects

Projects in Java with Tips:-

1. Desktop Quiz App
Let users choose a Quiz Topic (make at least 3), ask them questions and offer answers and then display the result. You can also include a certificate or display what questions they did good or not.
Tip: You can use JavaFX or Swing.

2. Chat App
Create a simple client-server application and allow clients to chat inside a group. So all clients send messages to the server, and server display them.
Tip: You can create it inside a console.

3. Password generetor with GUI
Create a password generetor where users can choose the length and password strength(e.g include numbers or not etc.).
Tip: You can use JavaFX or Swing.

4. Tic Tac Toe in Java
Create Tic Tac Toe in Java with GUI.
We are using Swing for GUI because of the simplicity of this project. No installation is required. You only need an IDE and Java.
Tip: I use Ecplise.
